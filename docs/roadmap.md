# Future Roadmap

## Near-term

- Structured output schemas for memory extraction and conversation classification
- More robust outbound rewrite pass when safety blocks content
- Admin-side editing for existing rows rather than append-only forms
- Better media reuse decisions for MMS

## Voice realism

- Publicly accessible generated speech assets for Twilio `<Play>`
- Realtime voice mode with a lower-latency speech stack
- Custom voice support with explicit consent workflow

## Multi-party support

- Family/operator notes and alerts UI
- Escalation routing by severity
- Trend views for mood, frequency, and missed messages
